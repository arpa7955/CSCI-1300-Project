------------------------
HOW TO COMPILE AND RUN
------------------------
Compile: g++ -Wall -Werror -Wpedantic -std=c++17 driver.cpp Board.cpp Player.cpp CandyStore.cpp Game.cpp
Run: ./a.exe
------------------------
DEPENDENCIES
------------------------
board.h, player.h, game.h, candyStore.h, characters.txt, candy.txt, and riddles.txt must be in the same
directory as the cpp files in order to compile.

ABOUT THIS PROJECT
------------------------
This project implements an interactive board game fashioned after the game
Candy Land. The user plays the role of one of the characters in a quest 
through the vast Candy Land world to reach the castle. As the
player travels through Candy Land, they encounter special tiles, hidden 
treasures, calamities, and more!
